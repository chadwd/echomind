Make two changes to `apps/web/src/components/InputPane.vue`:

**1. Card variant — from outlined to flat with M3 surface fill:**

Change:
```html
<v-card variant="outlined" rounded="lg" class="pa-6 h-100">
```
To:
```html
<v-card variant="flat" rounded="lg" class="pa-6 h-100" style="background-color: rgb(var(--v-theme-surface-container-low)); border: 1px solid rgb(var(--v-theme-outline-variant));">
```

This replaces the generic outlined border (which was using the old grey theme) with a warm purple-tinted fill (`#FCF1F8`) plus a subtle M3-tinted border (`#CFC3CD`). The card now reads as a recessed sidebar panel.

**2. GET FEEDBACK button — add elevation for confidence:**

Change:
```html
<v-btn
  color="primary"
  size="large"
  block
  :loading="isValidating"
  @click="$emit('validate')"
>
  Get Feedback
</v-btn>
```
To:
```html
<v-btn
  color="primary"
  size="large"
  block
  elevation="2"
  :loading="isValidating"
  @click="$emit('validate')"
>
  Get Feedback
</v-btn>
```

The `elevation="2"` gives the primary CTA a subtle lift that signals it's the action to take — distinguishing it from the flat disabled inputs above it.

**Do not change:** VSelect, VTextField configurations, card padding, layout structure, copy/labels (per CONTEXT.md constraint).
