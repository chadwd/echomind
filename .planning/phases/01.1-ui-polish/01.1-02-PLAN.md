Update the logo `<img>` element in `apps/web/src/App.vue` to eliminate the white square artifact and show only the icon mark portion of the PNG. The wordmark "EchoMind" stays as the `v-app-bar-title` text.

**Implementation approach:**

Use a fixed-size container with `overflow: hidden` and adjust `object-position` to frame the icon head/brain mark in the upper-left of the PNG. The icon portion of `logo-echomind.png` is at the top-left (the head/brain mark comes before the wordmark). Crop to show only that region.

Replace the current `<img>` style with:

```html
<img
  src="/logo-echomind.png"
  alt="EchoMind"
  style="
    display: block;
    height: 40px;
    width: 40px;
    object-fit: cover;
    object-position: left top;
    border-radius: 8px;
    margin-left: 16px;
    background: transparent;
  "
/>
```

Also update the app bar elevation and add a subtle bottom border to separate it from the content:

```html
<v-app-bar elevation="0" color="surface" border="b">
```

This removes the generic Material shadow in favor of a clean 1px divider (more tool-like, less app-bar-y). The `border="b"` prop adds Vuetify's standard bottom border using the `outline` color token.

Update the `v-app-bar-title` to use `text-subtitle-1 font-weight-bold` with letter-spacing for a tighter, more designed wordmark feel:

```html
<v-app-bar-title class="text-subtitle-1 font-weight-bold" style="letter-spacing: 1px; text-transform: uppercase; color: rgb(var(--v-theme-on-surface));">
  EchoMind
</v-app-bar-title>
```
